# TaskSoft

_Este repositório é uma restauração do meu TCC original desenvolvido em 2016/2017._

## O que é?
Projeto com o código original do meu TCC de 2016/2017, realizado no curso Técnico de Informática da ETEC Hortolândia, desenvolvido em C# e MySQL para um sistema de gestão de tarefas.

## Por que este repositório existe?
O projeto ficou guardado por mais de 8 anos em um pendrive que já apresentava mau contato pelo tempo, e hoje não funciona mais nas versões atuais quando tento executar.  
Por isso, decidi subir no GitHub para eternizar esse projeto e fazer com que ele funcione novamente.

Decidi subir e revisar o código para:
- Reaprender C# e relembrar como era minha lógica de programação na adolescência
- Atualizar dependências e referências
- Fazer o sistema rodar novamente nas versões modernas
- Registrar essa evolução do projeto que marcou meu início na TI

## Nota final
A ideia é que, ao final da refatoração, este README traga um passo a passo completo para que o **Matheus (eu)  do futuro** consiga rodar o projeto facilmente e reviver suas nostalgias.
